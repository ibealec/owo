// HOW IT WORKS

Three steps. Zero friction.

01
DESCRIBE

Tell owo what you need in plain English. No syntax to remember, no man pages to read.

02
GENERATE

AI converts your words into the perfect shell command, with full context of your environment.

03
EXECUTE

Review, edit if needed, and run — right in your terminal. Commands auto-add to shell history.

// FEATURES

Small tool. Big power.

6 AI PROVIDERS

OpenAI, Claude, Gemini, GitHub Models, Ollama/LM Studio, and Claude Code. Your choice.

CONTEXT-AWARE

Understands your OS, shell, directory structure, and recent commands for precise results.

SHELL HISTORY

Commands auto-added to history. Optionally include recent history for better context.

EDIT BEFORE RUN

Modify any generated command in-shell before executing. Full control, always.

CLIPBOARD READY

Auto-copy to system clipboard. Cross-platform support for macOS, Linux, and Windows.

EXPLAIN MODE

Use the -e flag to get plain-English explanations of what commands actually do.

// SEE IT IN ACTION

Real commands. Real results.

Find large files eating your disk $ owo find all files larger than 100MB → find / -type f -size +100M 2>/dev/null
Quick archive of the current directory $ owo compress this directory into a tar.gz → tar -czf archive.tar.gz .
See what's hogging space $ owo show disk usage sorted by size → du -sh * | sort -rh
Hunt down zombie processes $ owo find all zombie processes → ps aux | awk '$8=="Z" {print}'
// INSTALL

Get started in one command.

npm install -g owo-cli

Requires Node.js 18+. Then run owo setup to configure your AI provider.

brew install ibealec/owo/owo-cli

Available on macOS and Linux via Homebrew. Then run owo setup to get started.

git clone https://github.com/ibealec/owo.git && cd owo bun install && bun run build chmod +x dist/owo && mv dist/owo /usr/local/bin/owo

Requires Bun runtime. Then run owo setup to configure.
